It is necessary to install BDE library for correct function dBUSE programm.
(Run setup.exe in directory BDE\DISK1).
If library BDE had already instaled,
 it is sufficient to copy dBUSE directory content only.


For correct indication of national characters do following:    
1. Run dBUSE.exe
2. Select (by means of button Browse) file *.tbk
   in menu Tables/Options (in box BS100 code table). 
   Confirm with button SET.
3. Exit dBUSE programm.
